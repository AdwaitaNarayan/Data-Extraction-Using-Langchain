# Data Extraction Using LangChain

An AI-powered document processing application that extracts structured information from invoices, receipts, scanned PDFs, and other documents using LangChain, OCR, and Large Language Models.

## Features

- Extract data from invoices and receipts
- Support for scanned PDF documents
- OCR-based text extraction
- Structured JSON output
- Document classification
- Modular extraction pipeline

## Installation

Clone the repository:

```bash
git clone https://github.com/AdwaitaNarayan/Data-Extraction-Using-Langchain.git
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
